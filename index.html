Sentiment Analysis Tool - Free!

Input your text to generate the sentiment – positive, negative or neutral – with our Sentiment Analysis Tool.

Creating an account allows you to upload Excel files to generate sentiment.

You can either paste sentences directly or upload a dataset file (CSV / Excel).

Example:
I absolutely love this product!
It’s amazing!
I’m really disappointed with the quality of this item.

After upload, the server will run sentiment analysis on each row and return a file with predictions.
